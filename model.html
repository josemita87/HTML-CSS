Basic elements of HTML programming.

An HTML element is much more than a tag, as it is made up of:

  • An opening label.
  • Zero or more attributes.
  • Text enclosed by the label.
  • A closing tag

New tags were created in order to indicate mainly to search engines what the main, navigational and contextual content is. In this way we help to improve indexing in case search engines do not detect the importance of each text on our website.
